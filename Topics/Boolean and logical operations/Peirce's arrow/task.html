Peirce's arrow

In boolean logic, Peirce's arrow (also known as NOR) is a logical operator that produces a result that is the negation of the logical OR.

The operator returns true only if both arguments are false. Otherwise, it returns false. Kotlin doesn't have this built-in operator.

Select boolean expressions that implement the NOR boolean function. b1 and b2 are boolean variables that represent the arguments of the operator.
